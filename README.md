# pinit
A personal note-taking app.

> [!Note]
> Work in progress.. `pinit` is currently under development.

## Usage
```console
$ pinit --help
A personal note-taking app

Usage:
  pinit [flags]
  pinit [command]

Available Commands:
  configure   Setup

Flags:
      --help       Show help information
      --port int   port. Default: 3000 (default 3000)
      --version    Show version
```
